# SDK Guide

Official SDKs are planned but not yet public.

## Planned SDKs

- Python.
- JavaScript / TypeScript.

## Design Goals

- Typed request and response models.
- Explicit authentication configuration.
- Clear error handling.
- Compatibility with public OpenAPI contracts.
- No dependency on proprietary BCINexus or BCILattice source code.

## Temporary Approach

Until official SDKs are released, use REST examples and JSON schemas in this repository.
