# Plugin Guide

The BCINexus plugin system is planned for future public release. This repository includes a draft plugin manifest schema so developers can understand the intended integration shape.

## Planned Plugin Concepts

- Manifest-driven metadata.
- Explicit permissions.
- Versioned compatibility declarations.
- Public entry points.
- No access to proprietary internals.

## Draft Manifest

See [`../schemas/plugin-manifest.schema.json`](../schemas/plugin-manifest.schema.json).

## Compatibility

Plugin interfaces are not stable until a public plugin runtime is released.
