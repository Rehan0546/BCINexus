# Research Workflow Examples

BCINexus public examples focus on reproducible metadata and artifact exchange.

## Example Workflow Types

- EEG preprocessing metadata.
- BCI classification pipeline metadata.
- Pipeline import/export.
- Project publication metadata.
- Research artifact review.
- Plugin manifest planning.

## Good Public Artifact Practices

- Keep raw participant data out of public examples.
- Use synthetic or anonymized sample data.
- Record preprocessing assumptions.
- Separate training, validation, and test metadata.
- Include reproducibility metadata such as software versions and random seeds where applicable.

See [`../examples`](../examples) for placeholders.
