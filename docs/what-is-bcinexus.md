# What Is BCINexus?

BCINexus is the developer and cloud ecosystem around BCILattice. It provides the public surface for collaboration, publishing, reproducibility metadata, workflow exchange, API integrations, and future SDKs.

BCINexus is designed to help researchers and developers connect tools, automate workflows, and exchange reproducible artifacts without exposing proprietary product internals.

## Public Responsibilities

- Public API contracts.
- Integration examples.
- Authentication guidance.
- Workflow and pipeline interchange schemas.
- Public documentation and community support.
- Future SDK, plugin, and hardware integration guidance.

## Closed-Source Responsibilities

The commercial BCINexus service, hosted infrastructure, internal source code, private deployment configuration, and proprietary application logic are not included in this repository.
