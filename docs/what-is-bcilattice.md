# What Is BCILattice?

BCILattice is the local research application in the BCINexus ecosystem. It supports neuroscience and BCI-oriented workflows such as data organization, preprocessing, pipeline design, ML workflow planning, experiment tracking, analysis, reporting, and artifact export.

BCILattice is local-first. BCINexus complements it with cloud-facing capabilities such as account services, publishing, sharing, collaboration, and public developer integrations.

## Public Integration Boundary

This repository documents public interchange formats and integration patterns only. It does not include the BCILattice desktop application source code or internal implementation details.
